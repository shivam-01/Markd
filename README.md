# Markd

A modern markdown editor with live preview, themes, diagrams, and math support.

## Features

- Live preview with 8 themes
- Mermaid diagrams & KaTeX math
- Syntax highlighting (190+ languages)
- GitHub Flavored Markdown, alerts, footnotes
- Custom heading IDs, highlight syntax
- Drag & drop files, auto-save
- Zero dependencies — just HTML, CSS, JS

## Quick Start

```bash
# Just open in browser
open index.html

# Or use local server
./markd              # Linux
python -m http.server 8000   # Any platform
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + B` | Bold |
| `Cmd/Ctrl + I` | Italic |
| `Cmd/Ctrl + K` | Link |
| `Cmd/Ctrl + O` | Open file |
| `Cmd/Ctrl + S` | Download |
| `Cmd/Ctrl + E` | Themes |
